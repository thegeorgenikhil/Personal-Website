Beginners Guide on Writing Semantic HTML

20 July 2021

What is Semantic HTML?

Semantic HTML basically means writing HTML in way that gives meaning to each of our HTML elements in code. It clearly specifies the meaning of the content inside the tags to both the developer and the browser.

Examples of non-semantic elements: <div>,<span>

Examples of semantic elements: <form>,<aside>,<figure>

Semantic Elements in HTML

1. <section>

Section tags are used to define a particular section in a webpage. Inside the section tag we can group together contents of similar type/category.

2. <h1> to <h6>

Heading tags are used to convey headings. The heading tags give the text utmost importance than normal text elements. They also come with a default styling. The h1 to h6 tags are given according to their varying importance from top to bottom.

3. <article>

The article tag is used to represent individual sections in a webpage. Article tags are used mostly inside containers or sections. A given document can have multiple articles in it. One of the use cases of the article tag is in blogs, where each blog post can be enclosed in separate article tags.

4. <aside>

The aside tag is used to represent a particular section of text which may be indirectly related to the main content. Asides are frequently presented as sidebars or call-out boxes.

5. <header>

The header tags are used to represent introductory content in a page, for example: the hero section,which the user sees first in enclosed in the header tag. It may contain some heading elements but also a logo, a search form, an author name, and other elements.

6. <footer>

Similar to header the footer tag is used to represent the footer of a document or section. It may contain authorship information, copyright information, contact information , related-links etc.

7. <nav>

The nav tag is used to contain a section that help the user navigate the site by using navigation-links within. Only the important links are enclosed in the nav tag. Common examples of navigation sections are menus, tables of contents, and indexes.

8. <figure>

A figure element is used to contain images, illustrations. etc.The <figcaption> element is used to add a caption for the <figure> element
